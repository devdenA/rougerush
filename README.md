# RougeRush 

Application web fullstack de recherche de clubs de sport à Toulouse, 
avec un réseau social intégré pour les sportifs.

## Stack technique

- **PHP 8.2** — Architecture MVC, POO
- **MySQL 8.0** — Base de données relationnelle
- **Nginx** — Serveur web
- **Docker / Docker Compose** — Conteneurisation de l'environnement

## Lancer le projet

### Prérequis
- Docker Desktop installé

### Démarrage
```bash
docker-compose up --build
```
Puis ouvre http://localhost:8080

## Fonctionnalités (en cours de développement)

- Carte interactive des clubs de sport à Toulouse (Leaflet.js)
- Inscription et connexion sécurisée
- Feed social — publications, likes, commentaires
- Messagerie privée entre utilisateurs
